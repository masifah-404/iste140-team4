## Project Overview

Shattered Realms is a web-based community platform designed for Dungeons & Dragons players to connect and engage with one another. The goal of this project is to provide a centralized space where players can learn about the world, explore available classes, and become part of the Shattered Realms community.

## Features

- **User Registration & Profiles** — Players can create accounts and manage their presence within the community.
- **Classes** — Browse and explore the available D&D classes players can choose from within the Shattered Realms setting.
- **World Lore** — A dedicated section covering the lore and background of the Shattered Realms world.
- **Custom Merchandise** — Users can explore community merchandise tied to the Shattered Realms brand.

## Purpose

This project was built to support the growth of the Shattered Realms community by giving D&D enthusiasts the tools they need to stay connected and organized. It combines social features with practical game management utilities in one platform.
